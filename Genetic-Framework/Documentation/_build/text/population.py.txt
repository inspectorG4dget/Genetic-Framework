
population.py

genPop(N, chromGenfuncs, chromGenParams)
========================================

Return a population (list) of N unique individuals. Each individual
has len(chromgGenFuncs) chromosomes. For each individual, chromosome_i
is generated by calling chromGenFuncs_i(chromeGenParams_i)


genCharsChrom(l, chars)
=======================

Return a chromosome (list) of length "l", each of which is made up of
the characters from chars.
